# Day 1 — Exploratory Data Analysis (EDA) Mini Script

Part of my **#75DaysOfGitHub** challenge — daily commits building AI/ML skills and a public portfolio.

## What it does
- Loads a dataset into pandas
- Cleans it (removes duplicates, fills missing values)
- Prints summary statistics
- Generates a correlation heatmap (saved as PNG)

## Why
A solid EDA workflow is step 1 of every ML project. This script is a clean, reusable starting template for future projects.

## How to run
```bash
pip install pandas matplotlib seaborn
python eda_analysis.py
```

## Output
- Console: dataset shape, column info, summary stats
- File: `correlation_heatmap.png`

## Tech
Python, pandas, matplotlib, seaborn
